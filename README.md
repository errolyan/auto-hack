## 🖥️ 系统要求
要成功运行autohack，请确保你的系统满足以下要求：

- **操作系统：** Kali Linux 2020或更高版本
- **处理器：** 双核处理器或更好
- **内存：** 至少4 GB RAM
- **磁盘空间：** 至少200 MB可用空间
- **Python：** 必须安装3.6或更高版本

确保你的环境按照这些规范设置，以获得最佳体验。
